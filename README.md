# easy-goer

A small command line utility that fetches and installs Go binary releases.

Some ideas taken from [ruby-build](https://github.com/rbenv/ruby-build) and the likes.

## Basic usage

``` sh
./go-easy install 1.14.6 ~/go1.14.6
```
